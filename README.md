# daon 리뉴얼 프로젝트

기존 다온펜션 사이트를 리뉴얼한 프로젝트 입니다.

---

## 💡 주요기능 

* 펜션 소개 
* 예약 일부 구현 반응형 제작


## 🖥️ 사용한 기술 스택


* HTML
* JavaScript
* SASS

🔎 사용한 API 

* Swiper

## 📖 담당한 페이지


* 객실안내(Sub2)
* 이용후기(sub5)
* 이벤트(sub5)

⚙️ 담당한 기능

* 공통 모달 제작
