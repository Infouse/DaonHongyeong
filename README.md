## 📝 개요

* 기존 다온펜션 사이트를 리뉴얼한 프로젝트 입니다. 
* 노후화된 UI/UX를 전면 개편하여 다지인과 사용감을 개선하려 하였습니다.
* 반응형 웹을 제작하여 다양한 환경에서도 사용자에게 일관된 경험을 제공하려 하였습니다.
